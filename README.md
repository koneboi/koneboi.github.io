# Genomics Portfolio

A simple static portfolio for showcasing genomics, bioinformatics, and computational biology work.

## Structure

```
├── index.html      # Work page (Projects, Research, Skills, Experience)
├── about.html     # About page (Bio, CV, Education, Contact)
├── css/
│   └── styles.css
├── docs/
│   └── cv.pdf     # Your CV
└── README.md
```

## Features

- **Work** page: Projects, Research, Skills, and Experience sections
- **About** page: Bio, CV download, Education, and Contact links
- Clean, responsive design with Bootstrap 5
- Two-menu navigation (Work / About)

## GitHub Pages Deployment

1. Push this repository to GitHub
2. Go to Settings → Pages
3. Set Source to "Deploy from a branch"
4. Select `main` branch and `/ (root)` folder
5. Save and your site will be live at `https://yourusername.github.io/repo-name/`

## Customization

- Edit `index.html` to update projects, research, skills, and experience
- Edit `about.html` to update your bio, education, and contact info
- Replace `docs/cv.pdf` with your own CV
- Update `css/styles.css` to change colors or styling
